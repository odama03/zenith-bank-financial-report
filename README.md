# Zenith Bank Plc — Financial Report Dashboard

## Overview
This project analyzes 36 months of financial data from Zenith Bank Plc 
covering 2022, 2023 and 2024 to track revenue trends, profitability, 
return on equity and loan quality over time.

## Tools Used
- Python
- Pandas
- Plotly
- Dash

## Dataset
36 rows of monthly financial data including Interest Income, 
Non-Interest Income, Operating Expenses, Loan Loss Provision, 
Return on Equity % and NPL Ratio %.

## Dashboard Features
- KPI cards showing Total Revenue, Net Profit, Avg ROE and Avg NPL Ratio
- Total Revenue Trend line chart
- Net Profit by Month bar chart
- Quarterly Revenue vs Expenses grouped bar chart
- Return on Equity % line chart
- NPL Ratio % line chart
- Total Assets Growth area chart
